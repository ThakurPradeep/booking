# Resource Booking System

Full-stack Resource Booking System (Backend: Node.js + Express, Frontend: React + TypeScript + Redux Toolkit). Database: MySQL/Postgres (database name: `booking`).

Quick start (backend):

1. cd backend
2. copy `.env.example` to `.env` and edit DB connection
3. npm install
4. node seed.js   # creates schema and seeds test data
5. npm start

Quick start (frontend):

1. cd frontend
2. npm install
3. npm run dev

DB credentials (example):
- user: root
- password: 12345678
